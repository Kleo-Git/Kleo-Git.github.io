# Project Overview
A project which used NLP techniques for the purposes of text classification.

# Purpose
The goal of this project was to build an automated text classification system capable of routing customer queries (e.g., billing, account recovery, technical support) to the correct departments.

# Methodology
The project involved building an end-to-end NLP pipeline using Logistic Regression and TF-IDF Vectorization. I conducted a comparative analysis between two specific preprocessing strategies:

Pipeline 1: Used Stemming and Uni-grams for a baseline approach.

Pipeline 2: Used Lemmatization and Bi-grams to preserve linguistic meaning and semantic context.
